**Objetivo del Bad Code Generator GTP:**  
El objetivo de este proyecto es desafiar a los estudiantes a identificar y transformar "código malo" en soluciones elegantes y funcionales mediante la aplicación de patrones de diseño y buenas prácticas de programación. A través de ejemplos reales de código desordenado o ineficiente, los estudiantes desarrollarán habilidades críticas para depurar, refactorizar y optimizar software, preparándose para enfrentar desafíos del mundo laboral. Esta experiencia no solo fomentará el pensamiento crítico, sino que también demostrará cómo un enfoque creativo y disciplinado puede convertir problemas complejos en oportunidades de aprendizaje 🚀💻.
